密碼離線庫 (Offline Password Vault)
這是一個極簡、美觀且安全性極高的單檔案網頁密碼管理工具。採用 莫蘭迪配色 (Morandi Color Palette) 設計，專為個人、工作及家庭密碼分類管理而生。

🌟 核心特色
完全離線化：所有數據皆儲存於瀏覽器的 LocalStorage，不經過任何伺服器，確保隱私不外洩。

軍規級加密：使用瀏覽器原生 Web Crypto API，結合 AES-GCM (256-bit) 加密與 PBKDF2 密鑰衍生技術。

優雅介面：基於 Tailwind CSS 打造，支援流暢的動畫、響應式佈局（手機/電腦皆適用）。

類 PWA 體驗：內建 Apple Touch Icon 動態生成，方便加入手機桌面作為 Web App 使用。

🛠️ 技術規格
1. 加密流程
主密碼驗證：使用者設定的主密碼不會被明文儲存，而是透過 PBKDF2 搭配隨機鹽值 (Salt) 生成加密密鑰。

工作階段安全：解鎖後的金庫密鑰僅存於 sessionStorage，分頁關閉即自動失效。

數據加密：每筆儲存的 JSON 數據皆經過 AES-GCM 加密，並附帶唯一 IV (初始化向量)。

2. 功能亮點
三層分類系統：預設「個人、工作、家人」三大類，並支援二級子分類。

安全工具：內建密碼強度偵測器與隨機密碼產生器。

備份與恢復：支援匯出/匯入 JSON 格式的備份檔案，方便跨設備遷移。

防 XSS 注入：所有顯示欄位皆經過 HTML Escape 處理。

🚀 如何使用
開啟程式：直接使用瀏覽器開啟 index.html。

設定主密碼：初次啟動請設定一組「主密碼」，請務必牢記，一旦遺失將無法解密任何數據。

管理密碼：點擊右下角 + 號即可新增紀錄。

快捷操作：點擊帳號或密碼區塊可直接複製到剪貼簿。

定期備份：點擊標題下方的「導出」按鈕將加密前的原始數據存檔（請妥善保存於安全處）。

🎨 設計細節
字體：使用了 Noto Sans TC 與 Noto Serif TC 的字體搭配，提升專業感與閱讀舒適度。

配色：

背景：#f2e9e4 (米色)

文字：#22223b (深灰藍)

重點色：#9a8c98 (灰紫色)

免責聲明：本工具為離線密碼庫，請確保您的個人電腦與備份檔案之安全性。若 LocalStorage 遭手動清除且無備份，數據將無法找回。
